Procesamiento de datos categricos en Python

Vamos a aplicar en Python como sera el proceso de trabajar con variables categricas para codificarlas numricamente. Hay diferentes convenciones para el caso de Pandas y Scikit Learn.

El tema es mtodos que nos permitan codificar variables categricas en variables numricas que despus podamos trabajar en modelos de Machine Learning.

Hay casos que podemos tratar variables numricas como si fueran categricas? Claro que se puede y la utilidad es que depende si aplica para el caso especifico o no.

Encoding

Categora Dummy One-hot
ingles [0,0] [1,0,0]
espaol [0,1] [0,1,0]
frances [1,0] [0,0,1]
"nan" ? [0,0,0]

Tenemos 3 categoras [ingles, espaol, frances], aqu solo requerimos de un vector de 2 nmeros para codificarlas.

Como se puede ver en la tabla, es una representacin compacta, pero que sucede cuando entra otra categora, con el mtodo de encoding "Dummy" no hay posibilidad de meter otra categora para representarla $\leftarrow$ esta es la debilidad del encoding "Dummy".

Qu pasa ahora con el "One-hot", lo que hacemos es:

  • voy a contar cuantas categoras tengo [3]
  • entonces la longitud del vector es igual al nmero de categoras que tengo.
  • Lo que vamos a decir es: siempre que aparezca un 1 en la primera posicin es porque existe la categora ingles y las dems categoras no estn, entonces las hacemos 0.
  • As sucesivamente, es la filosofa del one hot

Que pasa con las nuevas categoras? La va interpretar como $[0,0,0]$ as puede contemplar nuevas categoras.

One hot en Scikit learn

Aqu vemos la documentacin sobre el proceso de codificar variables categricas

In[]:
import sklearn.preprocessing as preprocessing

#Voy a crear un mtodo de codificacin de variables de texto
encoder=preprocessing.OneHotEncoder(handle_unknown='ignore')

Aqu lo que hace:

handle_unknown='ignore'

Me permite precisamente cuando lleguen valores que no estn en la categoras contempladas y que yo necesite codificarlo, el puede codificarlo como un vector de $0's$. Ahora este enconder tengo que ajustarlo a las categoras de mi dataset, en mi caso lo ajustare para 'engine_type'.

In[]:
#Ajustando el encoder a mis datos
encoder.fit(df_cars[['engine_type']].values)
